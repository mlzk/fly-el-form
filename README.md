# Fly-El-Form
element-plus的 JSON To Form 轻量化表单组件

doc文档: [https://mlzk.github.io/fly-el-form/](https://mlzk.github.io/fly-el-form/)


## Install

```bash

npm install fly-el-form --save

```

## Usage

```js
// main.js
// ...引入vue和iview之后
// 引入组件库
import FlyElForm from "fly-el-form";
import "fly-el-form/lib/style.css";

Vue.use(FlyElForm);
```
